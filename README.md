# Blog Map

An interactive travel blog built with React and Google Maps. Each trip is a collection of journal entries pinned to a map. Photos are served via Cloudinary.

Live example: [blog.stevensouto.com](https://blog.stevensouto.com)

Forked from [clemlatz/road-trips](https://github.com/clemlatz/road-trips).

---

## Setup

### 1. Clone and install

```bash
git clone https://github.com/your-username/blog-map.git
cd blog-map
npm install
```

### 2. Create a `.env` file

```bash
cp .env.example .env
```

Fill in the values:

| Variable | Where to find it |
|---|---|
| `GOOGLE_MAPS_API_KEY` | [Google Cloud Console](https://console.cloud.google.com/) → Credentials |
| `CLOUDINARY_CLOUD_NAME` | [Cloudinary Dashboard](https://cloudinary.com/) → Settings → Access Keys |
| `CLOUDINARY_API_KEY` | Same as above |
| `CLOUDINARY_API_SECRET` | Same as above |
| `SITE_URL` | Your deployed site URL, e.g. `https://yourname.netlify.app` |

> **Google Maps key:** Set the application restriction to *HTTP referrers* and add your domain. The Geocoding API is handled via the Maps JS SDK so no separate key is needed.

### 3. Upload your photos to Cloudinary

Photos should be uploaded maintaining this folder structure:

```
images/{trip-id}/photos/{photo-id}
```

A script is included to bulk-upload from `dist/images/`:

```bash
node upload-to-cloudinary.js
```

### 4. Run locally

```bash
npm start
```

Open [http://localhost:8080](http://localhost:8080).

### 5. Build for production

```bash
npm run build
```

## Adding content

All content lives in `src/trips/trips.json` — an array of trip objects.

### Trip

```jsonc
{
  "id": "scotland",           // used in the URL: /scotland/
  "title": "Scotland",
  "subtitle": "July 2024",
  "linkTitle": "Scotland, July 2024",  // label shown on the overview map
  "mapCenter": { "lat": 56.98, "lng": -4.89 },
  "mapZoomLevels": { "desktop": 7, "mobile": 6 },
  "entries": [ ... ]
}
```

### Entry

```jsonc
{
  "id": "1",                  // unique within the trip, used in the URL
  "title": "Holyrood Park",
  "date": "July 13",
  "content": "We hiked for hours...",

  // Use either coords OR address (address will be geocoded automatically)
  "coords": [55.945, -3.167],
  "address": "Holyrood Park, Edinburgh, Scotland",

  "photos": [ ... ]
}
```

### Photo

```jsonc
{
  "id": "A1",               // must match the filename uploaded to Cloudinary
  "caption": "Optional caption shown under the photo"
}
```

Photos are fetched from Cloudinary at:
`images/{trip-id}/photos/{photo-id}`

Thumbnails are auto-generated by Cloudinary (no separate upload needed).

## Deploying to Netlify

1. Push to GitHub
2. Connect repo to Netlify
3. Set build command: `npm run build`, publish directory: `dist`
4. Add all `.env` variables under **Project Configuration → Environment variables**
